For Xbox One and Xbox Series X|S.

Hint:

dpad = D-Pad (unpressed)
dpad_dn = D-Pad Down (pressed)
dpad_lf = D-Pad Left (pressed)
dpad_rt = D-Pad Right (pressed)
dpad_up = D-Pad Up (pressed)
menu = Menu (acts as Start in X360 games)
windows = View (acts as Back in X360 games)

Everything else is self-explantory.
