# Kunny Krysis

Kunny Krysis is a shitty modded version of Candy Crisis, itself is a port of the old Mac game [with the same name](https://candycrisis.sourceforge.net) to modern macOS, Linux, and Windows.

The game is basically a yet another Puyo Puyo clone, this time filled with brainrot, memes, unfunny gooner content, AI-generated slop assets and lots of FIFA soundtracks during it's golden age.

Kunny Krysis is part of the The TetFuck Box.
